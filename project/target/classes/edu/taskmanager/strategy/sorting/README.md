# Подпакет sorting
Интерфейс стратегии сортировки задач и его реализации.

`TaskSortingStrategy.java`     (Интерфейс стратегии сортировки списка задач)
`DueDateSortingStrategy.java`  (Класс реализующий сортировку по сроку выполнения)
`PrioritySortingStrategy.java` (Класс реализующий сортировку по приоритету)

Так же можно реализовать другие стратегии сортировки:
- `AlphabeticalSortingStrategy`
- `CreationDateSortingStrategy`

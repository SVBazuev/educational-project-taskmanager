# Пакет decorator
Декораторы для добавления функциональности к задаче без изменения класса Task.

- `TaskDecorator.java`          (Абстрактный класс)
- `LoggingTaskDecorator.java`   (Декоратор, который логирует все изменения задачи (вызовы сеттеров))
- `TaggedTaskDecorator.java`    (Декоратор, который добавляет задаче возможность иметь теги (если исходная модель их не поддерживала, или для добавления дополнительных тегов поверх существующих))

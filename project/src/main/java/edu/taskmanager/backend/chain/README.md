# Пакет chain (Цепочка обязанностей)
Реализация фильтрации задач через цепочку обработчиков.

`TaskFilter.java`      (абстрактный класс или интерфейс)
`FilterChain.java`     (класс управляющий цепочкой фильтров)
`StatusFilter.java`    (класс реализующий фильтрацию по статусу)
`TagFilter.java`       (класс реализующий фильтрацию по тегу)
`ProjectFilter.java`   (класс реализующий фильтрацию по проекту)

# Пакет proxy
Реализация паттерна Proxy для контроля доступа к сервису задач.

`TaskService.java`            (интерфейс)
`TaskServiceImpl.java`        (реальная реализация)
`TaskServiceProxy.java`       (защитный прокси)
`LazyLoadingTaskProxy.java`   (виртуальный прокси для подзадач)
> Этот класс будет представлять задачу, но подзадачи загружаются только при запросе.
